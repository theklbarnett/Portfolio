User Dashboard

Overview

This application allows for the creation of users/admins on a similar platform as some of the older social media websites. Users can register and interact with each other on a communal wall. Each user can post messages and comment on other posts. The first user to register is automatically registered as an admin and as an admin can create/edit/delete other users as well as make them admins.

Features
  • JQuery utilizing Ajax to delete comments/messages for a more seamless user experience
  • Validation of user credentials during login as well as qualifications for registration
  • Ajax preventation of registering without form completion
  • Semi-responsively designed (on most pages)

Ecommerce

Overview

Similar to other ecommerce site structures, the main page of the site displays an array of products offered by the company and offers a filter by category or search by name function. A temporary cart for unregistered users allows for checkout and payment (although for demo purposes payment processing is not incorporated into the design). The admin login page allows for administrators to review orders and add additional products.

Features
  • Python backend with a Django framework
  • Javascript pop up window for admins to edit and add new products
  • Filtering of products/orders by category and keyword reloads only a portion of the page to allow for faster generation of content
